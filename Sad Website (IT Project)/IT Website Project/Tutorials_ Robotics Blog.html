Python Pandas/Numpy Project

What the program creates:

I took a CSV File about Bees from Kaggle and fetched it in my program. From there I sorted it, fit a Linear Model to it and removed outliers. After the outliers were removed and a model was created, I made a scatterplot of the original data and overlaid the linear line on the data, printing the equation to the legend of the chart. Once again, this is a very brief summary. From there I made a residual plot of the Predicted Price Values subtracted from the Actual Price Values. Finally, I used iteration to create a 5 Number Summary (Minimum, First Quartile, Median, Third Quartile, Maximum) of Honey Production for each of the years, which I printed to the terminal for now. In the future, I plan to turn these into parallel boxplots. A larger image of the scatterplot with regression line is here, and a larger image of the residual plot is here.
